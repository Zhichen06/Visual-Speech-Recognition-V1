# 视语灵枢 (VSR-Spiritual-Pivot): 中文唇语识别系统

本项目针对中文视觉语音识别（Visual Speech Recognition, VSR）任务，构建了一个端到端的唇语识别管线。项目目前处于中期阶段，已完成从原始视频预处理、口部区域自动裁剪到 3D 卷积 + TCN 模型训练的闭环。

---

## 📁 文件清单 (File List)

本项目共包含以下核心文件：

1.  **`中期报告.pdf`**: 项目中期技术总结报告。详细记录了系统架构、数据处理细节、分阶段训练实验分析（Loss 与 Accuracy 曲线）以及后续的技术转型计划。
2.  **`crop_mouth.py`**: 口部区域预处理脚本。利用 MediaPipe 或 RetinaFace 检测人脸关键点，并根据关键点从原始视频中裁剪出唇部序列，解决背景、头部姿态不一带来的噪声问题。
3.  **`dataset.py`**: 数据加载模块。定义了 `LRW1000Dataset` 类，负责读取索引文件、截取视频帧、灰度化、缩放（96x96）及 [0,1] 归一化。同时实现了随机水平翻转的数据增强逻辑。
4.  **`model.py`**: 模型定义文件。核心架构为 **3D Convolution Front-end + Temporal Convolutional Network (TCN)**。包含针对通道维度的“强制换维锁”逻辑，以适配 PyTorch 3D 卷积的输入规范。
5.  **`train_final.py`**: 自动化训练管线。支持断点续训（Checkpoint 自动恢复）、变长序列对齐（`vsr_collate`）、异常样本过滤以及多指标（Top-1, Top-5）日志记录。

---

## 🛠️ 技术路线 (Technical Specs)

- **数据集**: CAS-CSR-W1k-Dataset (LRW1000)。
- **视觉前端**: 1 层 3D 卷积层 (Kernel Size: 5x7x7)，用于捕捉短时空特征。
- **时序建模**: 4 层一维 TCN (Temporal Convolutional Network) 模块。
- **输入规范**: 灰度视频序列，单帧尺寸 96x96。
- **训练优化**: AdamW 优化器，学习率 1e-4，支持通过补零对齐处理变长序列。

---

## 🚀 核心功能

- **自动化预处理**: 实现了从原始视频到对齐唇部视频的自动化转换。
- **鲁棒性训练**: 引入随机水平翻转增强，并在 `forward` 过程中处理维度不一致问题。
- **断点续训**: 具备完善的 Checkpoint 管理机制，能够应对长周期训练中的中断。
- **异常容错**: 在 DataLoader 层级自动剔除损坏或读取失败的视频样本。

---

## 📈 阶段性成果

- **收敛性**: 模型在 1000 类中文词汇任务上表现出稳定的收敛趋势。
- **稳健性**: 实验证明，引入数据增强策略后，验证集准确率曲线由“锯齿状”波动转为单调平稳攀升，显著抑制了过拟合。
- **技术展望**: 已确定后续将向更高效的轻量化前端架构转型，以进一步提升模型的特征空间表达能力。
